#  Game Project

Para ejecutar el juego Piedra, Papel o Tijera sigue las instrucciones en la terminal:

```sh
cd game
python3 main.py
```

#  Gráfico pastel

#  population of countries

Genera graficas de pastel y de barras para ver la poblacion de diferentes paises/continentes. Dataset World Population:

```sh
cd app
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
